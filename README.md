# Operational Analytics Power BI Dashboard

## Project Overview

This project was created as a hands-on Power BI portfolio project after completing the Maven Analytics Power BI Desktop for Business Intelligence course.

The objective was to transform operational and procurement-related data into an executive reporting dashboard that supports business performance monitoring, inventory visibility, revenue attribution analysis, and supplier mapping.

The project uses realistic operational data based on hospitality and inventory management processes but is presented from a business analytics perspective.

---

## Business Questions

- What is the total revenue generated during the reporting period?
- Which products contribute most to revenue?
- How much inventory value is currently held?
- Which suppliers are connected to specific ingredients?
- How much revenue can be directly attributed versus estimated?

## Challenges Solved

During development several practical issues were encountered:

- Incorrect DAX references
- Month sorting problems
- Relationship validation
- Inventory valuation calculations
- Revenue attribution logic
- Dashboard layout optimization

These issues were identified and resolved through iterative testing and validation.

The final dashboard provides a business-friendly reporting solution for operational performance monitoring and demonstrates practical Power BI skills relevant to entry-level Analytics, Procurement, Supply Chain, and Operations roles.
